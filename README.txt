

First run :
	sh reset.sh //to create neccessary files that act as databases
	sh compile.sh //to compile and create output files for server and client



Then run :
	./server //to start the server
	./client //to run client



Now, few steps to follow:

1. Sign up as admin and add trains.
2. Now you can sign up as Agent and Customer to book tickets.



NOTE: Agent can login to the same account from multiple terminals. But Admin and Customer cannot 	   do so.
